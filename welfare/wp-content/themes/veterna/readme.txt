=== Veterna ===
Requires at least: 6.2
Tested up to: 6.6
Requires PHP: 7.4

== Description ==
Veterna is a WordPress theme designed for pet rescue, animal welfare, shelters, fundraisers, wildlife conservation, NGOs, NPOs, and other activism websites. Fully compatible with the WordPress Block Editor, this theme offers a professional design with a 100% responsive layout and is easy to customize, so no coding is required.
